# Setup

# Install server dependencies
npm install

# Install client dependencies
cd client
### `npm install`

# In the project directory, you can run:
### `npm start`

# Run both Express & React from root
cd ..
### `npm run dev`

# Build for production
cd client
### `npm run build`

# To Test
cd client
### `npm run test`
